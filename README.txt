La funcionalidad de este programa es poder ingresar a el sistema una serie de autos usados comprado a ciertos precios, para luego ser vendidos a el mismo precio o superior, también se ingresa los vendedores los cuales realizarán las ventas de los autos y por la venta recibirán una comisión, al correr el programa encontrará un menú por el cual podrá navegar entre las distintas funciones que tiene el programa.
